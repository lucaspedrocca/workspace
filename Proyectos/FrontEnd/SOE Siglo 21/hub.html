Las instancias de evaluación ahora también incorporan tecnología

Para que puedas rendir de manera segura los exámenes, Siglo 21 desarrolló la nueva plataforma SOE. Este sistema incorpora la aplicación Klarway que permitirá una validación biométrica y probados mecanismos de seguridad para la validación de cada instancia evaluativa.

¡No olvides que para el uso del SOE debes descargar Klarway y registrarte! Hacelo antes del comienzo de clases, solo así podrás seguir rindiendo normalmente tus exámenes. Vas a necesitar creatividad, ingenio, imaginación y trabajo en equipo para poder resolver los problemas de lógica y enigmas dentro de la aventura que escojas.
